Event Listener Ejemplo

  • Eventos del ratón
    • click: Se activa cuando se hace clic con el ratón.
    • dblclick: Se activa cuando se hace doble clic con el ratón.
    • mouseover: Se activa cuando el puntero del ratón entra en un elemento.
    • mouseout: Se activa cuando el puntero del ratón sale de un elemento.
    • mousedown: Se activa cuando se presiona un botón del ratón.
    • mouseup: Se activa cuando se suelta un botón del ratón.
  • Eventos del teclado
    • keydown: Se activa cuando se presiona una tecla.
    • keyup: Se activa cuando se suelta una tecla.
    • keypress: Se activa cuando se presiona una tecla (obsoleto en algunos navegadores).
  • Eventos de formularios
    • submit: Se activa cuando se envía un formulario.
    • change: Se activa cuando el valor de un input, select o textarea cambia.
    • input: Se activa cuando el valor de un input cambia en tiempo real.
    • focus: Se activa cuando un elemento obtiene el foco.
    • blur: Se activa cuando un elemento pierde el foco.
  • Eventos de ventana
    • load: Se activa cuando la página está completamente cargada.
    • resize: Se activa cuando se redimensiona la ventana del navegador.
    • scroll: Se activa cuando se desplaza el documento.
  • Eventos táctiles (para pantallas táctiles)
    • touchstart: Se activa cuando se coloca un punto de contacto en la superficie táctil.
    • touchend: Se activa cuando se retira un punto de contacto de la superficie táctil.
    • touchmove: Se activa cuando se mueve un punto de contacto a lo largo de la superficie táctil.
