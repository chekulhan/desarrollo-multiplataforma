# Actividad: “Crea tu personaje de videojuego”

Crear un personaje con atributos como nombre, edad, nivel, salud, clase, y poderes especiales, usando String, int, double, boolean y char. Luego lo imprimen en una tarjeta de personaje.

Por ejemplo, la interacción en la consola seria algo asi:

```bash
Ingresa el nombre de tu personaje: Arkan
Ingresa la edad del personaje: 25
Ingresa el nivel (1-100): 10
Ingresa la salud (puntos de vida): 75.5
¿Es un mago? (true/false): true
Ingresa la primera letra de tu clase (M = Mago, G = Guerrero, A = Arquero, etc.): M

=== PERSONAJE DEL VIDEOJUEGO ===
Nombre: Arkan
Edad: 25 años
Nivel: 10
Salud: 75.5
Mago? true
Clase: M
```
