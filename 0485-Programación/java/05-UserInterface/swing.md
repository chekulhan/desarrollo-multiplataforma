
# Java Foundation Classes
JFC (Java Foundation Classes) es un conjunto de bibliotecas de Java para crear interfaces gráficas de usuario (GUI) y manejar eventos en aplicaciones Java.
Fue introducido en Java 1.2 (1998) y sirve como base para Swing, que es la parte más conocida de JFC.

```bash

JFC (Java Foundation Classes)
│
├─ AWT (Abstract Window Toolkit)  → Componentes básicos dependientes del OS
├─ Swing                        → Componentes ligeros y personalizables
├─ PLAF (Look-and-Feel)         → Cambiar apariencia de la GUI
├─ 2D Graphics API              → Dibujos, gráficos, imágenes
├─ Accessibility API            → Soporte para usuarios con discapacidad
└─ Drag-and-Drop API            → Arrastrar y soltar objetos
```

En pocas palabras: JFC es la base completa para crear aplicaciones gráficas en Java, y Swing es su parte más usada hoy en día.
